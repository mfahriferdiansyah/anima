## Use Anima

For working in Anima day to day.

- [Getting started](/docs/use/getting-started): what Anima is, and the shape of your first session.
- [Notes](/docs/use/notes): capture, search, edit, and delete notes you own.
- [Publishing and export](/docs/use/publishing-and-export): publish one note as a link, or export the whole vault as markdown.
- [Companion](/docs/use/companion): the built-in agent that reads your notes and answers with citations.

## Build with Anima

For connecting your own agent to a vault.

- [Quickstart](/docs/build/quickstart): connect an agent over anima-mcp in about ten minutes.
- [MCP tool reference](/docs/build/mcp-reference): every tool your agent can call.
- [FAQ and troubleshooting](/docs/build/faq): the error messages you might hit, and how to clear each one.
- [Self-host Anima](/docs/build/self-hosting): run your own instance against a vault you already own.
- [How it works](/docs/build/concepts/custody-and-ownership): custody, the two-key model, Seal, Walrus, resurrection, and signed attribution.
