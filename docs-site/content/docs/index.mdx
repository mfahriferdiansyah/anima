Anima is an agentic workspace where your own AI agents and your team read and write the same notes and canvas, sealed to storage you own, so it survives any app.

These docs come in two tracks: **Use Anima** for working day to day, and **Build with Anima** for connecting your own agent.

## Agent-readable

Every page is available as clean markdown: hit the copy-markdown button on any page, or add `.md` to its path under `/md` (for example `/md/build/quickstart.md`). The whole site is indexed for coding agents at [`/llms.txt`](/llms.txt) and [`/llms-full.txt`](/llms-full.txt).
